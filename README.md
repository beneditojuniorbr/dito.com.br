# Dito - Micro SaaS Platform 🚀

Dito is a modern, minimalist Micro SaaS focused on empowering infoproducers and software creators. Built with a design language inspired by digital banking giants like Nubank, it provides a fast, fluid, and premium experience.

## 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Style**: Custom Vanilla CSS (Design Tokens, Grid/Flexbox, Animations)
- **Logic**: Vanilla ES6+ JavaScript
- **Icons**: Lucide Icons
- **Fonts**: Inter (via Google Fonts)

## ✨ Features
- **Premium Dashboard**: Nubank-style horizontal cards for real-time balance and sales monitoring.
- **Product Versatility**: Sell Ebooks, Courses, Mentorships, and **Micro SaaS**.
- **SaaS Ecosystem**: A unique differentiator allowing users to sell recurring subscriptions for their own software within the Dito infrastructure.
- **Hall of Fame**: A gamified ranking system with real-time updates and performance medals (🥇 🥈 🥉).
- **Mobile First**: Fully optimized for Android, iOS, and Desktop.

## 🚀 How to Run
1. Navigate to the project directory.
2. Open `index.html` in any modern web browser.
3. Use the Login screen to access the Dashboard (Click "Entrar").

## 💳 The SaaS-in-SaaS Model
When you create a "Micro SaaS" product in Dito, you can:
- Define Monthly/Annual billing cycles.
- Integrate with WhatsApp for automatic access delivery.
- Manage your own user base directly from the Dito panel.
